Part 1.1: Finite Difference Operator

The gradient across the X dimension is convoluted with the 2D matrix [[1, -1]], which is this operation for each pixel: \( \text{Gradient}_{x}(i,j) = I(i,j+1) - I(i,j) \). The gradient across the Y dimension is convoluted with the 2D matrix [[1], [-1]], which is this operation for each pixel: \( \text{Gradient}_{y}(i,j) = I(i+1,j) - I(i,j) \).
